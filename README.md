# Blinkit Data Analysis with Power BI

This repository contains a detailed analysis of Blinkit (formerly Grofers) grocery data using Power BI. The analysis provides insights into customer purchasing behavior, sales trends, and other key performance indicators to better understand Blinkitâ€™s operations.

## Project Overview

The goal of this project is to leverage Power BI to visualize and analyze the Blinkit grocery dataset. The analysis focuses on understanding sales patterns, customer preferences, and performance metrics to derive actionable insights for business improvement.

### Key Objectives:
- Perform data cleaning and transformation to prepare the Blinkit dataset for analysis.
- Create interactive Power BI dashboards to visualize sales trends and customer behavior.
- Generate insights on item sales, outlet performance, and customer purchasing habits.

## Dataset

The dataset used in this project is available in the repository as [BlinkIt grocery Dataset.xlsx](https://github.com/5udeep-da5/Blinkit-Data-Analysis-with-PowerBI/blob/main/BlinkIt%20grocery%20Dataset.xlsx). It contains various attributes related to products, sales, and outlets.

### Key Dataset Features:
- **Product Information**: Item type, fat content, and more.
- **Sales Data**: Total sales by product and outlet.
- **Outlet Details**: Outlet type, size, establishment year, and location.

## Power BI Dashboard

The Power BI dashboard includes the following visualizations:
- **Sales by Outlet Type**: Breakdown of sales across different types of outlets (e.g., supermarket, grocery store).
- **Item Sales Performance**: Analysis of top-selling product categories such as snacks, frozen foods, and household items.
- **Outlet Location Insights**: Identifies sales distribution across different geographic locations.

## Tools and Technologies Used
- **Power BI**: For creating interactive data visualizations and dashboards.
- **Excel**: For initial data cleaning and structuring.
- **GitHub**: For version control and collaboration.

## How to Use

1. Download the dataset from the repository.
2. Open the Power BI file (`Blinkit Data Analysis with PowerBI.pbix`) to explore the interactive dashboard.
3. Explore key metrics and insights derived from the analysis.
